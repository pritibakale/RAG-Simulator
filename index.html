What this teaches

You can watch how raw text becomes chunks, how fake but intuitive embeddings are produced, how similarity scores are calculated, and how an LLM uses retrieved chunks as grounded context.

Important realism note

This is a conceptual simulator. Real embedding models use high-dimensional vectors, and real LLMs generate responses probabilistically. Here, the math is simplified so the process becomes visible.
